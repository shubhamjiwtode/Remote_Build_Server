GUI Instruction:-
*After auto execution click kill button to enable test box and start process button.

* Select the test files and test driver from list box.
(you can select multiple files)

* Click CreateRequest Button to Create a Test Request of selected files.

*To add test to a particular test request select testfile,testdriver and testrequest to which you have to add test and then click add button

*Enter the Number of Child process you require(limit is : 1 to 9 child Process).

*After entering value between 1 and 9, StartProcess is enabled.

*Select the Test Req from the SelectRequest List box and click SendRequest to send the request to child builder


*KillProcess Button kills child process, repo, test harness.

*Closing GUI window kills everything .
